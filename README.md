**Dashboard de Ventas**

Este proyecto presenta un dashboard interactivo desarrollado en Power BI para el análisis de ventas a nivel global, permite visualizar el desempeño comercial mediante indicadores clave, facilitando la toma de decisiones estratégicas.

***Objetivo***
Analizar las ventas totales, el comportamiento por producto, país y año, así como evaluar los tiempos de entrega para identificar las oportunidades de mejora.

***Indicadores Principales***
💰 Ventas Totales: $1.27 millones
👥 Clientes: 89 
🌎 Países: 21  

**El dashboard integra diferentes perspectivas del negocio:**
***Ventas por Producto***
Ranking del Top 8 productos más vendidos, identificación los productos con mayor contribución a los ingresos.
***Ventas por Año***
Evolución de ventas en los años 1996, 1997 y 1998, donde se observa el análisis de tendencias mensuales.
***Ventas por País***
Mapa interactivo con distribución geográfica de ventas, con identificación de mercados clave donde Europa y América destacan como principales regiones de venta.
***Tiempo de Entrega***
Comparación del desempeño logístico por año  con clasificación de tiempos de entrega (1, 4, 8, 16, 23 días) que presentan diferencias importantes entre años, lo que puede impactar la satisfacción del cliente

***Herramientas Utilizadas***
Power BI
DAX (Data Analysis Expressions)
Modelado de datos

**Cómo usar el proyecto**
Descarga el archivo .pbix
Ábrelo en Power BI Desktop
Interactúa con los filtros de:
1. País
2. Producto
Explora las visualizaciones
