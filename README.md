Este proyecto consiste en un dashboard interactivo de ventas diseñado para analizar, visualizar, y entender el comportamiento comercial de una empresa, ayudandonos a identificar las tendencias y evaluar el desempeño comercial, que facilita la toma de decisiones basadas en datos. 

El objetivo principal es transformar datos de ventas en información visual clara que facilite:
1. El análisis de los ingresos obtenidos 
2. La identificación de productos más vendidos
3. La comparación entre países
4. El seguimiento por periodos de tiempo

Funcionalidad estratégica :
1. Visualización de ventas totales
2. Exploración detallada de datos
3. Gráficos dinámicos e interactivos
4. Análisis por producto o categoría
5. Segmentación por país
